Adding panorama support to Kaltura

A Kaltura video embedded the normal way

This can just be a regular website... The video player below is the Kaltura HTML5 player.

This website is statically hosted! Thanks to Kaltura, no back end coding is needed at all!

Suggestions for speech recognition: suggest that you want to visit the window, kitchen, or server room...

Please allow the website to access your camera if you want to see head tracking in action...

Features of the panorama viewer

  • Load 360 degree panoramic videos into mediaspace using the normal tools. Use the standard cylindrical projection produced by most panoramic photography systems
  • Edit your metadata in Kaltura to specify field of view and voice links
  • Play panoramic videos in the Kaltura HTML5 player addin with a plugin. (Sorry but Flash is not supported)
  • The thumbnails are still usable (untransformed panoramic images)
  • You can navigate with the "W" (up) "A" (left) "S" (right) and "D" (down) keys.
  • Supports webcam based headtracking for changing perspective!
  • Expands to two full screen views. A regular full screen view and a Oculus RIft full screen view with full head tracking (requires the vr.js browser plugin)!
  • Supports speech recognition! Navigate between pages by using web speech recognition. Speech recognition is initiated by pressing the "z" key.
  • Speech commands are Kaltura metadata which can be edited using the mediaspace API.
  • Add a speech recognition button to the web page with this line of html:
    <button onClick="if(window.videoSpeechEngine) videoSpeechEngine.start();">Start Speech Recognition</button>
